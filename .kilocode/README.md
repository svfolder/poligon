# Документация для Kilo Code

Этот проект использует специальную структуру документации для оптимизации работы с Kilo Code.

## Структура документации

- `architecture.md` - Архитектурные принципы проекта
- `coding_standards.md` - Принципы написания кода
- `project_structure.md` - Структура проекта
- `configuration.md` - Описание конфигурационных файлов
- `tools.md` - Инструменты сбора документации

## Конфигурационные файлы

Проект использует иерархическую систему конфигурации:

1. `common/config/` - Общие конфигурации для всех приложений
2. `console/config/` - Конфигурации для консольного приложения
3. `backend/config/` - Конфигурации для бэкенд приложения
4. `frontend/config/` - Конфигурации для фронтенд приложения

## Инструменты сбора документации

Проект предоставляет специальные инструменты для сбора документации:

- `console/controllers/DocController.php` - Генерация документации для классов
- `console/controllers/CollectCodeController.php` - Сбор информации о коде
- `console/controllers/StructureController.php` - Экспорт структуры проекта
